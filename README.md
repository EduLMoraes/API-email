# API-email:

Criei para estudar como hospedar api mesmo. Nada explêndido ou complexo, portanto, é de uso livre, faça bom uso.

Para usá-la basta usar o link https://emailapi.up.railway.app/ e fazer um _get_ com o email que deseja validar,
a entrada, como verás no código, é assim:
```rs
#[get("/{email}")]
```
